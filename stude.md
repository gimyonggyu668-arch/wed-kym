# 시험 문제 핵심

## 객체지향(OOP) 4대 특징 
### 캡슐화 : 데이터(필드)와 기능(메서드)을 하나로 묶고 외부에서 접근하지 못하도록 보호하는 것 
/ private과 setter,getter을 활용.

### 상속 : 기존에 있던 클래스에서 자원(메서드나 여러 기능)을 재사용하여 새로운 자식 클래스를 만드는 것

### 다향성 : 하나의 객체나 메서드가 여러가지 형태를 가질수 있는 것 
/ ex) 오버로딩이나 오버라이딩으로 자주 활용
interface 에서 자주 씌임

### 추상화 : 복잡한 실체에서 공통적인 특징만 추출하여 정의하는 것
